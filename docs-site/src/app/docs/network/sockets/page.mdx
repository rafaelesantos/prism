Conexões WebSocket em tempo real com `PrismNetworkSocketClient`.

## Visão Geral

PrismNetwork fornece suporte a WebSocket via `PrismNetworkSocketClient` e `PrismNetworkSocketAdapter`. O padrão é similar ao HTTP: defina um endpoint, conecte e troque mensagens.

### PrismNetworkSocketEndpoint

Defina o endpoint WebSocket:

```swift
import PrismNetwork

struct ChatSocketEndpoint: PrismNetworkSocketEndpoint {
    var scheme: PrismNetworkScheme { .wss }
    var host: String { "ws.example.com" }
    var path: String { "/chat" }
    // ...
}
```

### PrismNetworkSocketCommand

Defina comandos para enviar ao servidor:

```swift
struct ChatCommand: PrismNetworkSocketCommand {
    let action: String
    let payload: Data?
}
```

### Conectar e Trocar Mensagens

```swift
let socketAdapter = PrismNetworkSocketAdapter()

// Conectar
let messageStream = try await socketAdapter.connect(to: endpoint)

// Receber mensagens
for await data in messageStream {
    let message = try JSONDecoder().decode(ServerMessage.self, from: data)
    print(message)
}

// Enviar comando
try await socketAdapter.send(command: ChatCommand(action: "ping", payload: nil))
```

### Fluxo de Dados

```text
App → send(command) → SocketAdapter → Servidor
Servidor → AsyncStream<Data> → SocketAdapter → App
```
