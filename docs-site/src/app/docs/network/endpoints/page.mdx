Defina endpoints HTTP tipados conformando `PrismNetworkEndpoint`.

## Visão Geral

`PrismNetworkEndpoint` é o protocolo central do PrismNetwork. Cada endpoint é um tipo Swift que encapsula todos os aspectos de uma requisição HTTP: scheme, host, path, method, headers, body, query items, timeout e cache.

### Endpoints Básicos

```swift
import PrismNetwork

struct UsersEndpoint: PrismNetworkEndpoint {
    var scheme: PrismNetworkScheme { .https }
    var host: String { "api.example.com" }
    var path: String { "/users" }
    var method: PrismNetworkMethod { .get }
    var headers: [String: String] { [:] }
    var body: (any Encodable)? { nil }
    var queryItems: [URLQueryItem]? { nil }
    var timeoutInterval: TimeInterval? { 30 }
    var cacheInterval: TimeInterval? { 300 }
}
```

### Endpoint com Parâmetros

Use propriedades armazenadas para parameterizar endpoints:

```swift
struct UserDetailEndpoint: PrismNetworkEndpoint {
    let userID: String

    var scheme: PrismNetworkScheme { .https }
    var host: String { "api.example.com" }
    var path: String { "/users/\(userID)" }
    var method: PrismNetworkMethod { .get }
    var headers: [String: String] {
        ["Authorization": "Bearer \(token)"]
    }
    var body: (any Encodable)? { nil }
    var queryItems: [URLQueryItem]? { nil }
    var timeoutInterval: TimeInterval? { nil }
    var cacheInterval: TimeInterval? { nil }
}

let endpoint = UserDetailEndpoint(userID: "123")
```

### Endpoint POST com Body

```swift
struct CreateUserEndpoint: PrismNetworkEndpoint {
    let user: User

    var scheme: PrismNetworkScheme { .https }
    var host: String { "api.example.com" }
    var path: String { "/users" }
    var method: PrismNetworkMethod { .post }
    var headers: [String: String] {
        ["Content-Type": "application/json"]
    }
    var body: (any Encodable)? { user }
    var queryItems: [URLQueryItem]? { nil }
    var timeoutInterval: TimeInterval? { nil }
    var cacheInterval: TimeInterval? { nil }
}
```

### Propriedades Computadas

`PrismNetworkEndpoint` fornece automaticamente:
- `url` — `URL` completa construída a partir de scheme, host e path
- `request` — `URLRequest` pronta para uso com URLSession

### PrismNetworkMethod

| Método | Valor |
|--------|-------|
| `PrismNetworkMethod/get` | GET |
| `PrismNetworkMethod/post` | POST |
| `PrismNetworkMethod/put` | PUT |
| `PrismNetworkMethod/patch` | PATCH |
| `PrismNetworkMethod/delete` | DELETE |

### PrismNetworkScheme

| Scheme | Uso |
|--------|-----|
| `PrismNetworkScheme/http` | HTTP simples |
| `PrismNetworkScheme/https` | HTTPS (recomendado) |
| `PrismNetworkScheme/wss` | WebSocket seguro |

### PrismNetworkHeaderType

`PrismNetworkHeaderType` define tipos de header comuns (autorização, content-type, etc.).
