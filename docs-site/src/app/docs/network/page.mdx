Infraestrutura de rede HTTP e WebSocket com endpoints tipados, caching e logging integrado.

## Visão Geral

PrismNetwork fornece uma camada de rede type-safe sobre `URLSession`, com suporte a HTTP e WebSocket. O padrão central é o `PrismNetworkEndpoint` — um protocolo que define todos os aspectos de uma requisição (scheme, host, path, method, headers, body, query items, timeout, cache).

### Uso Básico

```swift
import PrismNetwork

// 1. Definir um endpoint
struct UsersEndpoint: PrismNetworkEndpoint {
    var scheme: PrismNetworkScheme { .https }
    var host: String { "api.example.com" }
    var path: String { "/users" }
    var method: PrismNetworkMethod { .get }
    var headers: [String: String] { [:] }
    var body: (any Encodable)? { nil }
    var queryItems: [URLQueryItem]? { nil }
    var timeoutInterval: TimeInterval? { 30 }
    var cacheInterval: TimeInterval? { 300 }
}

// 2. Fazer a requisição
let adapter = PrismNetworkAdapter()
let response = try await adapter.request(on: request, with: nil)
```

### WebSocket

Para conexões em tempo real, use `PrismNetworkSocketClient` com `PrismNetworkSocketEndpoint` e `PrismNetworkSocketCommand`.
