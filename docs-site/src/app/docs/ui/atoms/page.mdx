Componentes atômicos: os blocos fundamentais do design system.

## Visão Geral

Átomos são os menores componentes do PrismUI. Cada um é independente, reutilizável e acessível por padrão.

### PrismText

Exibe texto com suporte automático a skeleton loading e acessibilidade:

```swift
// Texto simples
PrismText("Olá, mundo")

// Com testID para XCUITest
PrismText("Título", testID: "title_label", isHeader: true)

// Com acessibilidade builder
PrismText("Valor") {
    $0.label("Valor monetário")
     .testID("amount_value")
}

// Com PrismResourceString
PrismText(AppString.welcomeTitle)
```

### PrismButton

Botão com feedback háptico, acessibilidade e testIDs:

```swift
// Texto simples
PrismButton("Entrar", testID: "login_button") {
    await authService.login()
}

// Com role
PrismButton("Deletar", testID: "delete_btn", role: .destructive) {
    deleteItem()
}

// Com label customizado
PrismButton {
    PrismHStack {
        PrismSymbol("star.fill")
        PrismText("Favoritar")
    }
} action: {
    toggleFavorite()
}
```

### PrismSymbol

Ícones SF Symbols com suporte a tokens de design:

```swift
PrismSymbol("star.fill")
PrismSymbol("heart", size: .large)
```

### PrismTextField

Campo de texto com máscara e configuração customizável:

```swift
PrismTextField("Email", text: $email)
    .prismAccessibility(.textField("Campo de email", testID: "email_field"))
```

### PrismAsyncImage

Carregamento assíncrono de imagens com placeholder:

```swift
PrismAsyncImage(url: imageURL) {
    PrismText("Carregando...")
}
```

### PrismSpacer

Espaçador que respeita os tokens de design:

```swift
PrismSpacer()
PrismSpacer(length: .large)
```

### PrismSection

Seção com header e conteúdo:

```swift
PrismSection("Configurações") {
    SettingsContent()
}
```

### PrismLabel

Label com ícone e texto:

```swift
PrismLabel("Perfil", systemImage: "person.fill")
```

### Layout Atômico

- `PrismHStack` — Stack horizontal
- `PrismVStack` — Stack vertical
- `PrismZStack` — Stack em camadas
- `PrismAdaptiveStack` — Stack que muda de eixo por plataforma. Veja `AdaptiveUI` para detalhes.

### Listas

- `PrismList` — Lista estilizada
- `PrismLazyList` — Lista lazy com performance
- `PrismHorizontalList` — Lista horizontal com scroll

### Outros

- `PrismTabView` — Abas adaptativas
- `PrismShape` — Formas com tokens de radius
- `PrismPreview` — Preview wrapper
- `PrismView` — Protocolo base para views do Prism
