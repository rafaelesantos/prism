export const metadata = { title: "Acessibilidade" }

Sistema de acessibilidade com suporte a VoiceOver, testIDs e builder pattern.

## Visão Geral

O PrismUI integra acessibilidade em cada componente. O sistema oferece duas APIs: direta via `PrismAccessibilityProperties` e conveniente via `PrismAccessibility` factory methods.

### PrismAccessibilityProperties

Controle total sobre acessibilidade:

```swift
let props = PrismAccessibilityProperties(
    label: "Botão de login",
    hint: "Toque para fazer login",
    testID: "login_button",
    traits: [.isButton, .allowsDirectInteraction]
)
```

Propriedades disponíveis:

| Propriedade | Tipo | Descrição |
|-------------|------|-----------|
| `label` | `String` | Rótulo VoiceOver |
| `hint` | `String` | Dica VoiceOver |
| `testID` | `String` | Identificador para XCUITest |
| `traits` | `[AccessibilityTraits]` | Traits de acessibilidade |
| `actions` | `[PrismAccessibilityAction]` | Ações customizadas |
| `inputLabels` | `[String]` | Labels de entrada |
| `value` | `String?` | Valor descritivo |
| `isHidden` | `Bool` | Se é visível para VoiceOver |

### PrismAccessibility Factory Methods

Métodos fábrica para roles comuns:

```swift
// Botão
PrismAccessibility.button("Entrar", testID: "login_btn")

// Campo de texto
PrismAccessibility.textField("Email", testID: "email_field")

// Header
PrismAccessibility.header("Configurações", testID: "settings_header", level: 1)

// Imagem
PrismAccessibility.image("Logo da empresa", testID: "logo_img")

// Texto
PrismAccessibility.text("Saldo disponível", testID: "balance_text")

// Grupo
PrismAccessibility.group(testID: "form_group", label: "Formulário de cadastro")

// Oculto do VoiceOver
PrismAccessibility.hidden(testID: "decorative_element")
```

### Builder Pattern

Use o builder pattern para acessibilidade em componentes:

```swift
PrismButton("Salvar") {
    saveAction()
} accessibility: {
    $0.label("Salvar alterações")
     .hint("Toque para salvar")
     .testID("save_btn")
}

PrismText("Valor") {
    $0.label("Valor da transação")
     .testID("amount_value")
}
```

### PrismAccessibilityAction

Ações customizadas acessíveis via VoiceOver:

```swift
let deleteAction = PrismAccessibilityAction.delete {
    deleteItem()
    return true
}

let adjustAction = PrismAccessibilityAction.adjust {
    adjustValue()
    return true
}
```

Ações disponíveis:
- `.delete(handler:)` — Ação de deletar
- `.adjust(handler:)` — Ação de ajustar valor
- `.expand(handler:)` — Ação de expandir
- `.collapse(handler:)` — Ação de colapsar
- `.custom(name:handler:)` — Ação customizada

### Integração com Componentes

Todos os componentes Prism aceitam acessibilidade:

```swift
// PrismButton com testID
PrismButton("Entrar", testID: "login_btn") {
    login()
}

// PrismText com header
PrismText("Título", testID: "title_label", isHeader: true)

// Modificador prismAccessibility
anyView
    .prismAccessibility(.button("Ação", testID: "action_btn"))
```
