Como adicionar o Prism ao seu projeto.

## Visão Geral

O Prism é distribuído como um Swift Package. Siga os requisitos e instruções abaixo para configurar.

### Requisitos

| Requisito | Versão Mínima |
|-----------|---------------|
| Swift | 6.3 |
| Xcode | 26.4 |
| iOS | 26 |
| macOS | 26 |
| Mac Catalyst | 26 |
| tvOS | 26 |
| watchOS | 26 |
| visionOS | 26 |

- Tipos de deployment intencionalmente seguem a última geração do SDK porque o `PrismUI` adota capacidades recentes do SwiftUI.

### Adicionar via Xcode

1. Abra o projeto no Xcode
2. Vá em **File → Add Package Dependencies**
3. Cole a URL do repositório
4. Selecione a versão ou branch desejada
5. Adicione os produtos necessários ao target

### Adicionar via Package.swift

```swift
dependencies: [
    .package(url: "<repository-url>", branch: "main")
]

targets: [
    .target(
        name: "YourApp",
        dependencies: [
            .product(name: "Prism", package: "prism")
        ]
    )
]
```

### Importar Módulos

Use o módulo umbrella para acessar tudo:

```swift
import Prism
```

Ou importe apenas os módulos que você precisa:

```swift
import PrismFoundation  // camada base
import PrismNetwork     // rede HTTP/WebSocket
import PrismArchitecture // store, reducer, router
import PrismUI          // design system
import PrismVideo       // download de vídeo
import PrismIntelligence // ML/AI local e remoto
```

- Tip: Importar módulos individuais reduz o tempo de compilação e deixa explícitas as dependências do seu target.
