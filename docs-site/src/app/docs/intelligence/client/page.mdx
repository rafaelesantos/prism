Fachada unificada para inteligência: local, Apple e remoto via `PrismIntelligenceClient`.

## Visão Geral

`PrismIntelligenceClient` é a API recomendada para consumo de inteligência no app. Abstrai os backends local, Apple Intelligence e remoto com a mesma interface.

### Backend Local

Use modelos CoreML treinados localmente para classificação e regressão:

```swift
import PrismIntelligence

// Com modelo existente
let local = try await PrismIntelligenceClient.local(modelID: "support-intent")
let label = try await local.classify(text: "Cobrar assinatura premium")

// Com modelo fornecido diretamente
let local = await PrismIntelligenceClient.local(model: myModel)
```

### Apple Intelligence

Use o modelo do sistema via FoundationModels (quando disponível):

```swift
let apple = PrismIntelligenceClient.apple()
let summary = try await apple.generate("Resuma os pontos principais da tela.")
```

Com configuração customizada:

```swift
let apple = PrismIntelligenceClient.apple(
    configuration: .init(
        model: .system(useCase: .general),
        instructions: "Responda de forma objetiva e segura."
    )
)
```

### Modelo Remoto

Conecte a provedores LLM via HTTP:

```swift
let remote = PrismIntelligenceClient.remote(
    endpoint: URL(string: "https://api.example.com/v1/generate")!,
    model: "gpt-oss-120b",
    headers: ["Authorization": "Bearer TOKEN"]
)

let answer = try await remote.generate("Crie um onboarding financeiro.")
```

Com serializer customizado:

```swift
let remote = PrismIntelligenceClient.remote(
    serializer: MyCustomSerializer(),
    transport: MyCustomTransport()
)
```

### Provider Genérico

Injete qualquer provider que conformo `PrismLanguageIntelligenceProvider`:

```swift
let client = PrismIntelligenceClient.provider(myProvider)
```

### Operações Disponíveis

| Método | Local | Apple | Remoto |
|--------|-------|-------|--------|
| `classify(text:)` | Sim | Não | Não |
| `classify(features:)` | Sim | Não | Não |
| `regress(features:)` | Sim | Não | Não |
| `generate(_:)` | Não | Sim | Sim |

### Status

Verifique a disponibilidade do backend:

```swift
let status = await client.status()
print(status.isAvailable)       // Bool
print(status.backend)           // .local, .apple, .remote
print(status.capabilities)      // [.classification, .regression]
print(status.modelName)         // "Support Intent v2"
```
