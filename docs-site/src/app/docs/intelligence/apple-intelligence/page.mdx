Integração com Apple Intelligence via framework FoundationModels.

## Visão Geral

`PrismAppleIntelligenceProvider` integra o Prism ao Apple Intelligence, usando o framework `FoundationModels` para acessar o modelo generativo do sistema.

### Uso via PrismIntelligenceClient

A forma mais simples de usar Apple Intelligence:

```swift
import PrismIntelligence

let client = PrismIntelligenceClient.apple()

let response = try await client.generate(
    "Resuma os pontos principais da tela inicial."
)
```

### Com Configuração

Use `PrismAppleIntelligenceConfiguration` para customizar o comportamento:

```swift
let client = PrismIntelligenceClient.apple(
    configuration: .init(
        model: .system(useCase: .general),
        instructions: "Responda de forma objetiva e segura."
    )
)
```

### PrismAppleIntelligenceModelReference

| Referência | Descrição |
|------------|-----------|
| `.system(useCase:)` | Modelo do sistema para um caso de uso específico |
| `.adapterName(_)` | Adapter de modelo customizado |

### PrismAppleIntelligenceUseCase

Casos de uso disponíveis:

```swift
PrismAppleIntelligenceUseCase.general    // Uso geral
PrismAppleIntelligenceUseCase.summarization  // Sumarização
// Outros use cases conforme SDK
```

### Uso Direto do Provider

Para controle mais fino, use o provider diretamente via `PrismLanguageIntelligence`:

```swift
let service = PrismLanguageIntelligence(
    provider: PrismAppleIntelligenceProvider(
        configuration: .init(
            model: .system(useCase: .general),
            instructions: "Responda de forma objetiva e segura."
        )
    )
)

let response = try await service.generate(
    .init(
        prompt: "Resuma os pontos principais da tela inicial.",
        context: [
            "App para iOS, macOS, tvOS, watchOS e visionOS",
            "Tom profissional"
        ]
    )
)
```

### Verificar Disponibilidade

```swift
let status = await client.status()
guard status.isAvailable else {
    print("Apple Intelligence indisponível: \(status.reason ?? "")")
    return
}
```

- Important: Apple Intelligence requer dispositivos compatíveis e configuração do sistema. Sempre verifique `status.isAvailable` antes de usar.
