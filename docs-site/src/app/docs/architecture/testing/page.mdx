Testes determinísticos com `PrismTestStore`.

## Visão Geral

`PrismTestStore` é um store de teste que permite verificar mudanças de estado de forma determinística, sem efeitos colaterais de timing.

### Criação

```swift
import PrismArchitecture

let testStore = PrismTestStore(
    initialState: CounterState(count: 0),
    reducer: CounterReducer()
)

// Com closure
let testStore = PrismTestStore(initialState: CounterState(count: 0)) { state, action in
    // reducer logic
}
```

### Enviar Ações e Verificar Estado

```swift
let newState = testStore.send(.increment)
XCTAssertEqual(newState.count, 1)

testStore.send(.decrement)
XCTAssertEqual(testStore.state.count, 0)
```

### Esperar por Effects

```swift
// Esperar que todos os effects completem
let completed = await testStore.waitForEffects(timeout: .seconds(2))
XCTAssertTrue(completed)

// Esperar até que uma condição seja verdadeira
let reached = await testStore.waitUntil { $0.isLoading == false }
XCTAssertTrue(reached)
```

### Com Middleware

```swift
let testStore = PrismTestStore(
    initialState: AppState(),
    reducer: AppReducer(),
    middleware: MyMiddleware()
)
```

### Integração com XCTest

`PrismTestStore` é projetado para funcionar com XCTest:

```swift
final class CounterTests: XCTestCase {
    func testIncrement() {
        let store = PrismTestStore(
            initialState: CounterState(count: 0),
            reducer: CounterReducer()
        )

        let state = store.send(.increment)
        XCTAssertEqual(state.count, 1)
    }

    func testAsyncEffect() async {
        let store = PrismTestStore(
            initialState: AppState(),
            reducer: AppReducer()
        )

        store.send(.loadData)
        let completed = await store.waitForEffects()
        XCTAssertTrue(completed)
        XCTAssertEqual(store.state.data.count, 10)
    }
}
```
