Armazenamento persistente chave-valor via UserDefaults com suporte a tipos `Codable`.

## Visão Geral

`PrismDefaults` é um wrapper sobre `UserDefaults` que serializa e desserializa tipos `Codable` automaticamente, removendo a necessidade de manipular `Data` ou `UserDefaults` diretamente.

### Uso Básico

```swift
import PrismFoundation

let defaults = PrismDefaults()

// Salvar
defaults.set("maria@email.com", for: "userEmail")

// Recuperar
let email: String? = defaults.get(for: "userEmail")
```

### Tipos Codable

Qualquer tipo `Codable` pode ser armazenado:

```swift
struct UserPreferences: Codable {
    let theme: String
    let notificationsEnabled: Bool
}

let prefs = UserPreferences(theme: "dark", notificationsEnabled: true)
defaults.set(prefs, for: "userPreferences")

let saved: UserPreferences? = defaults.get(for: "userPreferences")
```

### UserDefaults Customizado

Para testes ou isolamento, use o inicializador com `UserDefaults` customizado:

```swift
let testDefaults = PrismDefaults(userDefaults: UserDefaults(suiteName: "Test")!)
```
