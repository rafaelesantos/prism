Gerenciamento de arquivos com suporte a sandbox e privacidade.

## Visão Geral

`PrismFileManager` fornece acesso ao sistema de arquivos com abstrações para diretórios públicos e privados. `PrismFilePrivacy` define se um arquivo deve ser armazenado no diretório de documentos (público, visível no Files app) ou no diretório de suporte (privado).

### PrismFileManager

```swift
import PrismFoundation

let manager = PrismFileManager()

// Caminho público (visível no Files app)
let publicURL = manager.path(with: "report.pdf", privacy: .public)

// Caminho privado (não visível no Files app)
let privateURL = manager.path(with: "cache.db", privacy: .private)

// Tamanho do arquivo
let size = manager.size(at: publicURL) // "2.3 MB"
```

### PrismFilePrivacy

| Caso | Descrição |
|-----|-----------|
| `public` | Diretório de documentos — visível no Files app do iOS |
| `private` | Diretório de suporte — não visível para o usuário |

### Inicializadores

```swift
// Usa FileManager.default e diretório de documentos padrão
let manager = PrismFileManager()

// Com FileManager customizado (útil para testes)
let manager = PrismFileManager(fileManager: .default)

// Com URL de documentos customizada
let manager = PrismFileManager(fileManager: .default, documentsURL: customURL)
```
