# Quickstart

Build a complete **bookmark manager API** from scratch. By the end you'll have:

- HTTP server with JSON endpoints
- SQLite database with migrations
- Input validation
- CORS and logging middleware
- Error handling

All in pure Swift, zero dependencies.

<Info>
  Requires Swift 6.3+. Run `swift --version` to check. See [Installation](/installation) for setup details.
</Info>

## Create Your Project

<Steps>
  <Step title="Initialize a Swift package">
    ```bash
    mkdir bookmarks-api && cd bookmarks-api
    swift package init --type executable
    ```
  </Step>
  <Step title="Add Prism to Package.swift">
    ```swift title="Package.swift"
    // swift-tools-version: 6.0
    import PackageDescription

    let package = Package(
        name: "bookmarks-api",
        platforms: [.macOS(.v14)],
        dependencies: [
            .package(url: "https://github.com/byescaleira/prism.git", from: "4.4.0")
        ],
        targets: [
            .executableTarget(
                name: "bookmarks-api",
                dependencies: [
                    .product(name: "PrismServer", package: "prism")
                ]
            )
        ]
    )
    ```
  </Step>
  <Step title="Build to fetch dependencies">
    ```bash
    swift build
    ```
    First build takes a moment — subsequent builds are fast.
  </Step>
</Steps>

## Write Your Server

Replace `Sources/main.swift` with the following.

### 1. Server and Database

```swift title="Sources/main.swift"
import PrismServer
import Foundation

let server = PrismHTTPServer(port: 8080)
let db = try PrismDatabase(path: "bookmarks.db")

let migrator = PrismMigrator(database: db)
try await migrator.migrate([
    PrismMigration(
        name: "create_bookmarks",
        up: """
            CREATE TABLE bookmarks (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                title TEXT NOT NULL,
                url TEXT NOT NULL,
                tags TEXT DEFAULT '',
                created_at TEXT DEFAULT (datetime('now'))
            )
            """,
        down: "DROP TABLE bookmarks"
    )
])
```

<Tip>
  Use `":memory:"` for an in-memory database during development. See [Database overview](/server/database/overview) for connection options.
</Tip>

### 2. Middleware

```swift
await server.use(PrismCORSMiddleware())
await server.use(PrismLoggingMiddleware())
await server.use(PrismErrorMiddleware(includeStackTrace: true))
```

Middleware runs on every request, in registration order. The logging middleware wraps everything after it, capturing total response time.

For all available middleware, see the [Middleware reference](/server/middleware/cors).

### 3. List Bookmarks

```swift
await server.get("/bookmarks") { request in
    let rows = try await db.query("SELECT * FROM bookmarks ORDER BY created_at DESC")
    let bookmarks = rows.map { row -> [String: String] in
        [
            "id": "\(row["id"]!)",
            "title": "\(row["title"]!)",
            "url": "\(row["url"]!)",
            "tags": "\(row["tags"]!)",
            "created_at": "\(row["created_at"]!)"
        ]
    }
    return .json(bookmarks)
}
```

Every route handler receives a [`PrismHTTPRequest`](/server/core/request) and returns a [`PrismHTTPResponse`](/server/core/response). The `.json()` factory serializes any `Encodable` value and sets the right headers.

### 4. Create a Bookmark (with Validation)

```swift
await server.post("/bookmarks") { request in
    let validation = request.validateJSON { v in
        v.field("title", .required, .minLength(1), .maxLength(200))
        v.field("url", .required, .url)
    }

    if let errorResponse = validation.errorResponse() {
        return errorResponse
    }

    guard let body = request.body,
          let json = try? JSONSerialization.jsonObject(with: body) as? [String: Any] else {
        return PrismHTTPResponse(status: .badRequest, body: .text("Invalid JSON"))
    }

    let title = json["title"] as? String ?? ""
    let url = json["url"] as? String ?? ""
    let tags = json["tags"] as? String ?? ""

    try await db.execute(
        "INSERT INTO bookmarks (title, url, tags) VALUES (?, ?, ?)",
        parameters: [.text(title), .text(url), .text(tags)]
    )

    let id = await db.lastInsertID
    return .json(["id": id, "created": true], status: .created)
}
```

Validation rules compose naturally. `.required` ensures the field is present, `.url` checks for valid format. See [server validation](/server/security/validation) for all built-in rules.

### 5. Get a Single Bookmark

```swift
await server.get("/bookmarks/:id") { request in
    let id = request.parameters["id"]!

    guard let row = try await db.queryFirst(
        "SELECT * FROM bookmarks WHERE id = ?",
        parameters: [.text(id)]
    ) else {
        throw PrismAppError.notFound("Bookmark not found")
    }

    return .json([
        "id": "\(row["id"]!)",
        "title": "\(row["title"]!)",
        "url": "\(row["url"]!)",
        "tags": "\(row["tags"]!)",
        "created_at": "\(row["created_at"]!)"
    ])
}
```

Route parameters (`:id`) are extracted via `request.parameters`. `PrismAppError.notFound()` is caught by [`PrismErrorMiddleware`](/server/core/error-handling) and converted to a 404 JSON response.

### 6. Delete a Bookmark

```swift
await server.delete("/bookmarks/:id") { request in
    let id = request.parameters["id"]!
    let changes = try await db.execute(
        "DELETE FROM bookmarks WHERE id = ?",
        parameters: [.text(id)]
    )

    guard changes > 0 else {
        throw PrismAppError.notFound("Bookmark not found")
    }

    return .json(["deleted": true])
}
```

### 7. Start the Server

```swift
print("Bookmarks API running on http://localhost:8080")
try await server.start()
try await Task.sleep(for: .seconds(.max))
```


```bash title="Get One"

```

```bash title="Delete"
curl -X DELETE http://localhost:8080/bookmarks/1
```

```bash title="Validation Error"
curl -X POST http://localhost:8080/bookmarks \
  -H "Content-Type: application/json" \
  -d '{"title": "", "url": "not-a-url"}'
```
</CodeGroup>

Validation error returns structured feedback:

```json
{
  "errors": {
    "title": ["is required"],
    "url": ["must be a valid URL"]
  }
}
```
